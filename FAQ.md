# FAQ

## Comment changer mon mot de passe ?

La fonctionnalite de changement de mot de passe n'est pas encore disponible. Contacte un administrateur.

## Je ne peux pas voir les snippets apres connexion

Verifie que tu es bien connecte. Si le probleme persiste, deconnecte-toi et reconnecte-toi.

## Pourquoi je ne peux pas reviewer mon propre snippet ?

Par conception, un etudiant ne peut pas reviewer son propre code. Cela garantit des retours objectifs.

## Pourquoi je ne peux pas voter pour ma propre review ?

Le systeme de vote est reserve aux pairs pour eviter les abus.

## Mes badges ne sont pas attribues

Les badges sont attribues automatiquement apres chaque action (creation de review, vote). Rafraichis la page ou verifie ton profil.

## Un snippet inapproprie a ete poste

Utilise le systeme de signalement sur les reviews concernees. Les moderateurs traiteront le signalement.

## Comment devenir moderateur ?

Seul un administrateur peut promouvoir un utilisateur moderateur. Contacte ton enseignant.

## L'editeur de code ne s'affiche pas

L'editeur utilise Monaco (VS Code). Assure-toi d'avoir une connexion internet pour charger les assets.

## J'ai oublie mon mot de passe

Contacte un administrateur pour reinitialiser ton mot de passe. La fonctionnalite de reinitialisation sera ajoutee dans une version future.

## Les langages disponibles

Voir la liste complete sur la page de creation de snippet. Pour en ajouter, contacte un administrateur.
